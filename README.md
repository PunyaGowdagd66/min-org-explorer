# 🌐 Org Explorer

Org Explorer is a simple web application that allows users to search GitHub organizations and view their statistics such as:
- Number of repositories
- Number of followers
- Total stars
- Organization profile link

It uses the GitHub REST API to fetch live data and display it in a clean and user-friendly interface.

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- GitHub REST API
- GitHub Pages (for deployment)

## 📸 Features

- 🔍 Search GitHub organizations by name  
- ⭐ Sort repositories by stars  
- 📊 View repository count, followers, and total stars  
- 🌙 Clean and simple UI  
- 🔗 Direct link to GitHub organization
